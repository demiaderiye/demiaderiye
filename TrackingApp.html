Barcode Tracking app

This app allows buisnesses to track order production as it moves along the manufacturing process.

First the user is prompted to enter the assembly barcode.

User is prompted to enter their name.

Displays the data saved to the databaes. From here the user can change the user name or end the assembly.

If they choose to change employee it will return them to the employee page. Otherwise they can end assembly and are prompted to start a new assembly. Which will take them back to the user page.

This is a simple version of a tracking app. Depending of your companies needs features can be added like pausing, adding multiple employees, multiple departments etc... This app can be as robust or simple as needed. Feel free to check out the php and html used to create this app with the button below
